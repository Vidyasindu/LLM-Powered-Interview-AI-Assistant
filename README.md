### 🤖 LLM Powered Interview AI Assistant 🧠

LLM Powered Interview AI Assistant is an AI-driven interview preparation platform built using Streamlit, LangChain, and Large Language Models (LLMs). The system helps users prepare for technical interviews by generating intelligent interview questions, AI-powered answers, explanations, and technical guidance in real time.

---

## ✨ Features

* 🤖 AI-powered interview question generation
* 📄 Resume-based interview analysis
* 🧠 Technical explanations with LLMs
* 💻 Python coding assistance
* 📊 Domain-specific interview preparation
* 🎯 Real-time AI guidance
* 🔍 Keyword extraction and analysis
* 🌐 Streamlit interactive UI

---

## 🛠️ Tech Stack
